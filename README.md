## StellarPay

A non-custodial payment and asset platform built on Stellar.

## Features
- Wallet creation
- XLM payments
- Custom asset issuance
- DEX price viewer
- Soroban escrow contract

## Tech Stack
- Python
- React
- Stellar SDK
- Soroban
